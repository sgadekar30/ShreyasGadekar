Intelligent Refrigerator to Recommend Recipes.

Intelligent Refrigerator to Recommend Recipes using Hybrid Recommendation approach

The basic idea behind the project was to "Recommend Recipes" to the customers based on items present inside their Refrigerator. In addition to this, the system also recommended recipes based on ratings given by similar users. For doing so, we used content-based and collaborative filtering. The full report for the project can be found here

Working of project:
The system would first scan the recipes inside the refrigerator using a PiCamera attached to a Raspberry Pi and then send the image to the Google Firebase(where all data is stored) using a Flask Server. When the user needs a recommendation, the Django server fetched the images, classifies them using Tensorflow, and then creates a list of items. This list is then used to recommend recipes to the user. If the user wants to see some new recipes that similars users had, there is an option for that as well.

SMAFTY - Smart + Safety

SMAFTY - Smart + Safety

This was a project done for Smart India Hackathon which was then "The Biggest Hackathon". Me and my team had reached the Grand Finale of the hackathon as well.

Idea behind the project:
Due to increasing crimes against women, we came up with the idea of designing and developing a Safety device to load them with a powerful yet easy, small, and handy arsenal to make them feel safe. The task was to design and develop a which can be easily mounted on our body and would be easy to access quickly. The product was typically intended for Women Safety but could be used by anyone.

Working of project:
The system hardware consisted of an Arduino Mega with GPS, GSM with Sim and Bluetooth capabilities. In addition to this, there is an Android Application as well. The steps followed by the system are as follows:

  1. Setting up the system:
    • Turn the device on and start the Bluetooth, GSM, and GPS modules.
    • Connect the android device and the safety device via Bluetooth.
    • Store the emergency contact numbers of family and friends into the application.

  2. What happens in case of an emergency when the user presses the Panic button on Device: :
    • The device fetches the current location of the user using the GPS module
    • Send the location to the emergency helpline numbers(like 911 in the US, 100 for police in India)
    • Send the signal to the android device which then sends the location in a text message to the numbers stored while setting up the device.
    • Send a notification to all the people having the application in a radius of 5km.

The functionality of sending a notification to users nearby was used so that anyone living nearby can reach the spot before waiting for the police to come to the scene and save the victim.

AI Learns to play Flappy Bird using Genetic Algorithm

AI Learns to play Flappy Bird using Genetic Algorithm

Here, is the link to github repo. for this project.
This was a project which focused on implementing NEAT to make the AI learn the game of flappy bird.

The algorithm first generates 10 bird objects and passes them to a neural net with no hidden layers. The input takes 3 values, a bias term, the distance between the top pipe and the bird, and the distance between the bottom pipe and the bird. Then, it works around with random values of weights and then check the performance of the birds. The best ones are selected for the next level and mutated to get another generation. This is the way the system finds the weights and decides whether to jump or not.

Since the system is very easy, the AI sometimes learns very quickly. However, in some cases, the system fails to come up with good values and ends up repetitively generating weak generations. To tackle this, the system terminates after 50 generations. There are 2 types of AI shown below:
